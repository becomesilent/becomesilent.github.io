Hexo 初识

2020-08-09

简介

Github Pages + Hexo 搭建一个个人学习记录的博客
Github Pages 是一种免费托管在Github上的静态网页
Hexo是一个快速简洁生成博客的一个框架，提供各种不同的主题，可以选择自己喜欢的主题生成静态网页

开始使用

创建仓库

首先在Github官网创建一个仓库,
仓库的名字必须为xxx.github.io, 其中xxx是你Github的用户名

安装Node.js 和 Git

官网下载， 傻瓜式安装
